1) CD to this folder
2) run: java TTTO.TTTOnlineStartup
3) Start a MasterServer via the GUI
4) Start at least two by running "java TTTO.TTTOnlineStartup" again and select "Play TTT"
4) Connect to the masterserver with the clients *


*) If default IP is wrong when joining with a client, wait a while and you will be able to set which ip it should connect to.
   (connect to the IP displayed in the MasterServer GUI)
